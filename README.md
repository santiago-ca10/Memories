# Cartas para Ti 🎓

Una pequeña carta digital interactiva, hecha para celebrar una graduación.
Desliza las tarjetas como si pasaras recuerdos y descubre el mensaje final.

## ✨ Características

- Pantalla de bienvenida animada
- Tarjetas deslizables (mouse, touch, tap o botón "Siguiente")
- Fondo degradado pastel con partículas brillantes
- Confeti al llegar al final
- Música ambiental suave, activa desde el inicio (generada, sin archivos de audio)
- Botón para reiniciar y volver al inicio

## 📁 Estructura del proyecto

```
Memories/
│
├── index.html          # Estructura principal
├── css/
│   └── style.css        # Estilos y animaciones
├── js/
│   └── script.js         # Lógica de tarjetas, confeti y música
└── assets/               # Espacio para imágenes o audio propio (opcional)
```
## 🛠️ Cómo correrlo localmente

Solo abre `index.html` en tu navegador, o usa una extensión tipo Live Server.
